# Module 1: 90-Day Launch Plan

## The Master Roadmap

A day-by-day timeline starting 12 weeks before launch. Every task in order with time estimates: building your ARC team, setting up your KDP listing, stacking reviews, and executing launch week hour by hour.

---

### Phase 1: Foundation (Weeks -12 to -9)

**Goal:** Infrastructure before content

| Week | Tasks |
|------|-------|
| -12 | Set up author website/landing page. Create email list (MailerLite free tier). Research 20 ARC readers in genre communities. |
| -11 | Write the free lead magnet (prequel chapter, bonus epilogue, or checklist). Design cover for lead magnet in Canva. |
| -10 | Start building ARC reader list (see Module 2). Set up StoryOrigin/BookSirens profiles. |
| -9 | First ARC recruitment posts in r/selfpublish, genre Discords, Facebook groups. Aim for 60+ expressions of interest. |

### Phase 2: Build the Machine (Weeks -8 to -5)

**Goal:** Everything ready before launch

| Week | Tasks |
|------|-------|
| -8 | Write and polish your Amazon blurb using Module 3 prompts. Research 7 backend KDP keywords. Select KDP categories. |
| -7 | Format manuscript for KDP (Kindle Create or Atticus). Design paperback cover. Write acknowledgments and author bio. |
| -6 | Set up pre-order on KDP. Finalize ARC reader list — send 20–30 advance copies. Write your 6-email launch sequence (Module 4). |
| -5 | Follow up with ARC readers (reminder: reviews go live launch week). Write newsletter swap emails. Reach out to 5 genre newsletters. |

### Phase 3: Warm-up (Weeks -4 to -1)

**Goal:** Momentum before the button

| Week | Tasks |
|------|-------|
| -4 | Start social media content (Module 5). Post 2x/week. Schedule launch week posts. |
| -3 | Send newsletter swaps. Emails should go to THEIR list this week — reciprocate next. Run first "cover reveal" post. |
| -2 | Final ARC follow-up: "Reviews go live [date] — please post by [date+3]." Check KDP pre-order dashboard. |
| -1 | Everything scheduled. 3 social posts queued per day for launch week. Launch day email written. Backup plan for any last issues. |

### Phase 4: Launch Week (Week +1)

**Goal:** Reviews → rank → sales flywheel

| Day | Tasks |
|-----|-------|
| Mon | **LAUNCH DAY.** Book goes live. Email your list. Post on all socials. Message ARC team to post reviews. |
| Tue | Thank ARC posters individually. Monitor rank. Post organic content (not salesy). |
| Wed | Newsletter swap partner sends. Engage every comment. Post user-generated content. |
| Thu | Second email to list (social proof angle). Check Amazon rank trends. |
| Fri | Thank you post: "X reviews in Y days!" Email with early reader quotes. |
| Sat | Low-key. Engage social comments. Prepare next week's content. |
| Sun | Weekly review. Log numbers. Plan week 2 adjustments. |

---

## Time estimates for each phase

| Phase | Hours needed |
|-------|-------------|
| Foundation (weeks -12 to -9) | ~10 hours |
| Build the machine (weeks -8 to -5) | ~15 hours |
| Warm-up (weeks -4 to -1) | ~8 hours |
| Launch week | ~5 hours |
| **Total** | **~38 hours** |
